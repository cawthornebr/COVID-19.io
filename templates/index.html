Covid-19 - Confirmed cases by county by day

-Source of data-

Johns Hopkins University (JHU): https://coronavirus.jhu.edu/map.html

JHU GitHub for COVID data - https://github.com/CSSEGISandData/COVID-19

JHU only started breaking cases down by county starting 3/22/2020, because of this, I'll only be using data from that date forward.

-Info-

The JHU site above only displays the total number of confirmed, deaths, and recovered cases. I wanted to know the rate that confirmed cases were increasing which isn’t displayed on their site; to me that’s all that matters at this point. Originally, I wanted to create charts that displayed the rate cases were increasing by day by county for Arizona. After writing a program that generated these charts for just Arizona, I had others ask me to do this for their state as well. I scaled my code so now it will generate a state specific csv and the accompanying charts.


Located at the top of this screen, select the state from the drop-down menu to display the rate increase by county.
